## О проекте

Проект представляет собой современную платформу для видеоконференций и
совместной работы, объединяющую в себе надежность корпоративных инструментов и
простоту массовых сервисов. Пространство, где дистанционное общение становится
максимально естественным: за счет использования передовых алгоритмов сжатия данных
пользователи получают стабильную картинку в высоком разрешении и чистый звук даже
при слабом интернет-соединении (работает даже на парковке). В основу системы заложен
принцип мгновенного доступа, позволяющий подключаться к звонкам в браузере в один клик,
без необходимости регистрации или установки тяжелого программного обеспечения.
